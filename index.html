Neochitary:

Neochitary es un bot de Discord multifuncional. Algunas de sus funcionalidades incluyen:

  • Moderación: Ayuda a mantener el orden en el servidor.
  • Juegos: Ofrece varios juegos para entretener a los miembros del servidor.
  • Música: Permite a los usuarios reproducir música en un canal de voz.
  • Imágenes: Proporciona imágenes y gifs para diversión.
  • Social: Ofrece comandos para interactuar socialmente con otros miembros del servidor.
  • Crypto: Proporciona información sobre criptomonedas.
  • Utilidades: Ofrece una serie de comandos útiles para diferentes propósitos.
  • Configuración: Permite a los administradores configurar ciertos aspectos del servidor.
  • IA: Ofrece comandos para interactuar con la inteligencia artificial.
  • Otros: Ofrece una variedad de comandos útiles para diversas tareas.
